# Projeto Calculadora

Este projeto é uma calculadora simples desenvolvida em C# utilizando conceitos de  Programação Orientada a Objetos (OOP).

# Funcionalidades:

A calculadora permite que o usuário insira dois números e um operador (+, -, *, /) para realizar uma operação matemática. Se o usuário inserir um operador inválido ou tentar dividir por zero, uma mensagem de erro será exibida.

# Instruções de uso:

1. Abra o projeto em seu ambiente de desenvolvimento de software
2. Execute o programa
3. Siga as instruções na tela para inserir os números e o operador
4. O resultado da operação será exibido na tela

# Limpeza da tela:

O usuário pode limpar a tela a qualquer momento digitando "limpar" no console.
